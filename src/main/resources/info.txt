    Приложение для транслирования траекторий из “HyperMill” в Siemens NX.
Для правильной работы приложения и сохранения порядка траекторий,
необходимо определять имена траекторий в алфавитном порядке.
Пример:
1-черновая
2-semiFinish
3-чисто.
И т.д.

    После окончания работы приложения рекомендуется обновить сессию NX.
Перейти в модуль моделирования (cntl + M), затем вернуться в модуль обработки (cntr + alt + M).

    Приложение содержит текстовое окно, в которое можно копировать пути к директориям “pof-файлов”,
вводить директории в ручную, или воспользоваться выбором через диалоговое окно “Выбрать директорию”.
    В случае если в пути имеется ошибка, текстовое окно изменяет свой цвет на красный.
После исправления ошибки, текстовое окно снова становиться белым.

    Приложение не поддерживает направление вращения шпинделя против часовой стрелки.
Направление вращение шпинделя всегда будет транслироваться по часовой стрелки.

	Приложение не поддерживает экспорт траекторий 3-Axis с коррекцией на радиус инструмента.

    Приложение поддерживает только 2 типа инструментов (фрез):
- End Mill (концевая фреза без углового радиуса, или с угловым радиусом)
- Ball Mill (цилиндрическая , или коническая фреза со сферическим торцом)

    Имя СКС в HyperMill не должны совпадать с именами СКС в NX.
